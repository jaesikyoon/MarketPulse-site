# MarketPulse-site

MarketPulse 브리핑 배포 전용 레포지토리.

GitHub Pages를 통해 매일 아침 미국 증시 모닝 브리핑을 제공합니다.

## 사이트

https://jaesikyoon.github.io/MarketPulse-site/

## 구조

- `index.html` - 아카이브 메인 페이지
- `latest-audio.mp3` - 최신 음성 브리핑 (매일 덮어쓰기)
- `YYYY/MM/DD/index.html` - 날짜별 브리핑 (영구 보관)

## 배포

[MarketPulse_v1](https://github.com/jaesikyoon/MarketPulse_v1) 레포의 GitHub Actions에서 자동 배포됩니다.
